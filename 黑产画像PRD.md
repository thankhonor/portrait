# 黑产画像 PRD

## 1. 产品概述

### 1.1 产品名称

黑产画像（Gang Profile）

### 1.2 所属模块

AI引擎运营平台 > 业务反欺诈

### 1.3 产品定位

黑产画像是面向反欺诈运营人员的情报分析工具，用于检索、查看和分析黑产账号及群组的画像信息，辅助运营人员了解黑产组织结构、风险特征和关联线索，为线索跟进决策提供参考依据。

### 1.4 目标用户

反欺诈运营人员

### 1.5 入口位置

侧边栏导航：业务反欺诈 > 黑产画像

路由地址：`/antifraud/gang-profile`

---

## 2. 功能架构

```
黑产画像
├── 检索列表页
│   ├── 左侧：账号/群组检索面板
│   │   ├── 关键词搜索
│   │   └── 来源渠道筛选
│   └── 右侧：检索结果列表
│       └── 点击卡片 → 跳转详情页
│
├── 画像详情页
│   ├── 左侧：基本信息面板
│   │   ├── 账号画像（账号类型）
│   │   └── 群组画像（群组类型）
│   └── 右侧：关联信息面板
│       ├── 线索 Tab
│       ├── 群组 Tab（账号类型专有）
│       ├── IOC Tab（账号类型专有）
│       ├── 文件 Tab（账号类型专有）
│       └── 账号 Tab（群组类型专有）
│
└── 画像抽屉（详情页内使用）
    └── 点击群组/账号列表的查看图标 → 打开抽屉
```

## 3. 数据模型

### 3.1 画像类型

系统支持两种画像类型：

| 类型 | 标识 | 说明 |
|------|------|------|
| 账号 | account | 黑产个人账号画像 |
| 群组 | group | 黑产群组画像 |

### 3.2 账号画像字段

| ID | 账号唯一标识 | GANG001 |

## 4. 功能详述

### 4.1 检索列表页

#### 4.1.1 页面布局

页面采用左右分栏布局，整体以白色面板呈现：

- **左侧**：搜索面板（宽度 280px，浅灰背景）
- **右侧**：检索结果列表（自适应宽度）

#### 4.1.2 搜索面板

**搜索条件：**

| 条件 | 类型 | 说明 |
|------|------|------|
| 搜索内容 | 文本输入 | 支持按账号昵称、曾用昵称、账号ID 模糊搜索 |
| 来源渠道 | 下拉选择 | 选项：全部 / Telegram / WhatsApp / Discord / 暗网论坛 |

**操作按钮：**

- 搜索：执行搜索，支持回车键触发
- 重置：清空所有搜索条件

**结果统计：**

搜索面板底部显示"共找到 X 个账号"。

#### 4.1.3 检索结果列表

**默认展示：** 无搜索条件时，按最近活跃时间倒序展示 TOP 5 结果，不显示空态。

**搜索后展示：** 展示所有符合条件的结果，按最近活跃时间倒序排列。

**空态：** 无匹配结果时，显示搜索图标和"未找到匹配的账号或群组"提示。

**列表卡片信息：**

| 信息项 | 说明 |
|--------|------|
| 圆形头像 | 取名称首字，背景色按序轮换（蓝、紫、绿、橙、粉、青、靛） |
| 名称 | 画像名称 |
| 类型标识 | 蓝色"账号"标签 / 橙色"群组"标签 |
| ID | 画像 ID |
| 曾用昵称 | 仅账号类型显示，单行展示 |
| 风险场景 | 灰色标签，显示场景名称和命中次数 |
| 来源渠道 | 来源平台名称 |
| 最近活跃 | 最近活跃时间 |

**交互行为：**

- 悬浮卡片：背景变为浅蓝色，右侧出现箭头图标
- 点击卡片：跳转至画像详情页

---

### 4.2 画像详情页

#### 4.2.1 页面布局

- **顶部**："← 返回列表"导航按钮
- **主体**：左右双栏布局
  - 左栏：基本信息面板（宽度 520px，带边框）
  - 右栏：关联信息面板（自适应宽度，带边框）
- 两栏等高，内容可独立滚动

#### 4.2.2 左栏 - 基本信息面板

**面板标题：** 账号类型显示"账号画像"，群组类型显示"群组画像"，标题固定在顶部。

**画像信息：**

- 圆形头像 + 名称 + 外部链接图标
- ID 信息
- 基本字段信息（详见 3.2 / 3.3 节）
- 风险场景标签列表

#### 4.2.3 右栏 - 关联信息面板

**面板标题：** "关联信息"

**Tab 切换：** 采用胶囊式分段控件样式（灰色背景容器，选中项白色背景带阴影）。

**账号类型的 Tab：**

| Tab | 说明 |
|-----|------|
| 线索 | 关联的情报线索 |
| 群组 | 参与的群组列表 |
| IOC | 威胁情报指标 |
| 文件 | 关联的文件和图片 |

**群组类型的 Tab：**

| Tab | 说明 |
|-----|------|
| 线索 | 关联的情报线索 |
| 账号 | 群组中的关联账号 |

---

### 4.3 线索 Tab

#### 4.3.1 统计概览

以 7 列网格展示线索状态统计：

| 统计项 | 颜色 |
|--------|------|
| 线索总数 | 黑色 |
| 待审核 | 橙色 |
| 暂不关注 | 灰色 |
| 待跟进 | 蓝色 |
| 跟进中 | 绿色 |
| 跟进结束 | 紫色 |
| 已生成事件 | 红色 |

#### 4.3.2 风险分布图

SVG 环形图，展示各风险场景的线索数量分布，右侧显示图例（色点 + 场景名 + 数量）。

#### 4.3.3 线索列表

每条线索显示：

| 信息项 | 说明 |
|--------|------|
| 状态标签 | 彩色标签，对应6种状态 |
| 线索标题 | 线索内容摘要，超长截断 |
| ID | 线索编号 |
| 风险场景 | 所属风险场景 |
| 发布渠道 | 信息来源渠道 |
| 发布时间 | 线索发布时间 |

**交互：** 右侧查看图标，点击在新窗口打开线索详情。

---

### 4.4 群组 Tab

展示当前账号关联的群组列表。

每条群组信息：

| 信息项 | 说明 |
|--------|------|
| 群组名称 | 群组名称 |
| ID | 群组 ID |
| 成员数 | 群组成员数量 |

**交互：** 点击查看图标，打开该群组的画像抽屉。

---

### 4.5 IOC Tab

展示与当前画像关联的威胁情报指标。

**IOC 类型及标签颜色：**

| 类型 | 标签颜色 | 示例 |
|------|---------|------|
| IP | 蓝色 | 185.234.xx.xx |
| 域名 | 绿色 | creditking2024.cc |
| 邮箱 | 紫色 | dumps@proton.me |
| 手机号 | 橙色 | +86-159\*\*\*\*3421 |
| Hash | 灰色 | a3f2b8c1d4e5...9f0a |

每条 IOC 显示：类型标签、值、首次发现时间、最近发现时间。

---

### 4.6 文件 Tab

#### 4.6.1 文件筛选

提供两个圆角胶囊筛选按钮：

- **全部**：显示所有文件，括号内显示总数
- **图片**：仅显示图片类文件，括号内显示图片数量

选中状态为蓝色实心，未选中为白色描边。

#### 4.6.2 文件网格

以 3 列网格展示文件卡片。

**文件类别及标签颜色：**

| 类别 | 标签颜色 |
|------|---------|
| 图片 | 蓝色 |
| 文档 | 绿色 |
| 代码 | 紫色 |
| 数据 | 橙色 |

**每张卡片包含：**

- 缩略图区域（彩色占位背景 + 文件图标）
- 来源信息（如"来源群聊: 来源于发布内容"）
- 文件类别标签
- 上传时间
- "查看详情"链接

#### 4.6.3 悬浮交互

鼠标悬浮在缩略图区域时，出现半透明黑色遮罩层：

- **图片类文件**：显示"预览"和"下载"两个圆形白色按钮
- **非图片类文件**：仅显示"下载"按钮

#### 4.6.4 图片预览 Lightbox

点击图片的预览按钮后，打开全屏图片预览：

| 功能 | 说明 |
|------|------|
| 图片展示 | 居中大尺寸展示当前图片 |
| 关闭 | 右上角关闭按钮，或点击遮罩层关闭 |
| 左右导航 | 左右箭头切换上一张/下一张图片，边界时隐藏对应箭头 |
| 页码 | 底部显示"当前序号 / 总数" |
| 工具栏 | 底部圆形按钮组：缩小、放大、全屏、旋转 |

---

### 4.7 账号 Tab

展示当前群组中的关联账号列表。

每条账号信息：

| 信息项 | 说明 |
|--------|------|
| 账号名称 | 账号昵称 |
| ID | 账号 ID |
| 角色 | 在群组中的角色（如群主、成员、核心成员、数据供应商等） |

**交互：** 点击查看图标，打开该账号的画像抽屉。

---

### 4.8 画像抽屉

#### 4.8.1 触发方式

在画像详情页的群组 Tab 或账号 Tab 中，点击列表项的查看图标触发。

#### 4.8.2 抽屉规格

- 从右侧滑出
- 宽度：600px
- 覆盖在当前页面之上

#### 4.8.3 抽屉内容

**头部：**

- 圆形头像 + 名称 + 类型标签（账号/群组） + 外部链接图标
- ID 信息

**基本信息区域：**

- 标题栏带深色竖线装饰
- 展示完整的基本字段信息
- 风险场景标签

**关联信息区域：**

- 标题栏带红色竖线装饰
- 下划线式 Tab 切换（蓝色选中态）
- Tab 标签显示数量，如"线索 (6)"
- Tab 内容与详情页右栏一致

**账号类型 Tab：** 线索、群组、IOC、文件

**群组类型 Tab：** 线索、账号

## 5. 状态标识体系

### 5.1 线索状态

| 状态 | 中文名 | 标签颜色 |
|------|--------|---------|
| pendingReview | 待审核 | 橙色 |
| ignored | 暂不关注 | 灰色 |
| pendingFollowup | 待跟进 | 蓝色 |
| following | 跟进中 | 绿色 |
| followEnded | 跟进结束 | 紫色 |
| eventCreated | 已生成事件 | 红色 |

### 5.2 画像类型标识

| 类型 | 标签文本 | 颜色 |
|------|---------|------|
| 账号 | 账号 | 蓝色（文字蓝色、背景浅蓝、边框蓝色） |
| 群组 | 群组 | 橙色（文字橙色、背景浅橙、边框橙色） |

## 6. 页面导航流

```
检索列表页
  │
  │ 点击卡片
  ▼
画像详情页
  │
  ├── 线索 Tab → 点击查看图标 → 新窗口打开线索跟进页
  │
  ├── 群组 Tab → 点击查看图标 → 打开群组画像抽屉
  │
  ├── IOC Tab（仅查看，无跳转）
  │
  ├── 文件 Tab → 点击图片预览 → 打开 Lightbox
  │
  └── 账号 Tab → 点击查看图标 → 打开账号画像抽屉
  │
  │ 点击"返回列表"
  ▼
检索列表页
```

## 8. 非功能性说明

### 8.1 数据说明

当前版本使用前端 Mock 数据，共 7 条画像样本数据。后续需对接后端 API 接口。

### 8.2 技术实现

- 框架：Next.js App Router
- UI 组件：Radix UI + Tailwind CSS
- 图标库：Lucide React
- 图表：自定义 SVG 环形图
- 国际化：支持中/英文切换（通过 LanguageContext）
